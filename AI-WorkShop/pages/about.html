О проекте

Цель проекта

Создать и обучить VK-бота, способного вести осмысленный диалог с пользователями, отвечать на часто задаваемые вопросы и помогать в решении простых задач. Основной фокус - на автоматизацию общения в образовательных и небольших рабочих группах.

Технологический стек

  • C# (aiogram / C#-vk-bot)
  • OpenAI API / YandexGPT / GigaChat
  • PostgreSQL / SQLite для хранения истории диалогов
  • Docker для развёртывания
  • Git и GitHub для контроля версий

Этапы работы

  1. Анализ требований и выбор инструментов
  2. Создание базового функционала бота
  3. Интеграция с ИИ-моделью
  4. Тестирование и улучшение качества ответов
  5. Деплой и написание документации
